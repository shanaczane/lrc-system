About this book

Even bad code can function. But if code isn't clean, it can bring a development organization to its knees. Every year, countless hours and significant resources are lost because of poorly written code. This handbook presents a revolutionary paradigm with practical advice on writing readable, maintainable software — drawn from real-world examples by master craftsmen.
